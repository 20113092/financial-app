# Financial Management App

This is a Kotlin-based console financial tracking app that allows users to manage Incomes & Expenses, generate Financial Report and view balances over time. 
This App is useful to people who want to track their financials in an easy, basic but simple way.

## Features

- Add Income
- Add Expenses
- List all Income
- List all Expenses
- Generate Full Fianancial Report
- Generate Date Range Report
- Calculate OTtal Income, Expenses & Balance

  ## How to get started

  - Download ZIP
  - Extract Files
  - Open IntelliJ
  - Select File - Open
  - Chose the unzipped folder "financial-app"
  - Press the green go button at the top of IntelliJ
